# 🎮 Eğitici Oyun Tasarımı - CodeQuiz

## Açıklama
CodeQuiz, programlama bilgilerini eğlenceli bir şekilde test eden interaktif bir bilgi yarışması oyunudur. HTML, CSS, JavaScript ve Python kategorilerinde sorular içerir.

## Özellikler
- 🎯 4 farklı programlama kategorisi (HTML, CSS, JavaScript, Python)
- ⏱️ Zamanlı sorular (zorluk seviyesine göre 15-30 saniye)
- 🔥 Kombo sistemi (art arda doğru cevaplar için bonus puan)
- 📊 Puan sistemi ve istatistikler
- 🏆 Liderlik tablosu (localStorage ile kalıcı kayıt)
- 🎨 3 zorluk seviyesi (Kolay, Orta, Zor)
- ✨ Animasyonlu geri bildirim (doğru/yanlış)
- 🌙 Modern koyu tema tasarım

## Teknolojiler
- HTML5
- CSS3 (Animations, Glassmorphism, Neon effects)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## Kullanım
1. `index.html` dosyasını tarayıcıda açın
2. Kategori ve zorluk seviyesi seçin
3. "Oyunu Başlat" butonuna tıklayın
4. Soruları zamanında cevaplayın
5. Sonuçlarınızı kaydedin ve liderlik tablosunu takip edin!
