# 📱 Mobil Uygulama - EduLearn Mobile

## Açıklama
EduLearn Mobile, e-öğrenme platformu için tasarlanmış mobil öncelikli bir uygulamadır. Telefon çerçevesi simülasyonu ile gerçekçi bir mobil uygulama deneyimi sunar.

## Özellikler
- 📱 Gerçekçi telefon çerçevesi (iPhone tarzı)
- 🏠 Ana sayfa (kurs kartları, kategoriler, ilerleme çubukları)
- 🔍 Arama ve keşfet ekranı (canlı filtreleme)
- 📚 Kurs detay sayfası (hakkında, dersler, yorumlar)
- 👤 Profil sayfası (istatistikler, ayarlar)
- 🔔 Bildirimler ekranı
- ✨ Touch-friendly UI ve smooth animasyonlar
- 📊 Bottom navigation bar

## Teknolojiler
- HTML5
- CSS3 (Flexbox, Animations, Transitions)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## Kullanım
`index.html` dosyasını tarayıcıda açın.

## Ekran Görüntüleri
Mobil görünüm simülasyonu masaüstü tarayıcıda çalışır. 375px altı ekranlarda tam ekran olarak görüntülenir.
